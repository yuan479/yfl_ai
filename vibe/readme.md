## Vibe Coder
是最近在硅谷特别火的一个概念，一种全新的编程方式。

**氛围**编程？
对于不太会的知识点...
早先，百度查找
后来，通义千问，编辑器离开,离开当前编程？
当碰到一个bug？花点时间去解决。

cursor / Trae
VIbe coding 我们去全面拥抱相信大模型，沉浸式的开发产品。
我们不需要离开，不需要会太多代码，甚至不会代码，也可以协同各种背景的人，去开发一种产品，从而盈利。
想法是比代码更珍贵的东西，这是一种代码，学习和生活方式。

##必需
-AI编辑器
-idear 产品创意
-选择完全相信大模型
一个公司，一个人结合大模型可以完成一个产品，沉浸式

-可以与代码无关，有最好
-可以提高效率，快速拥有产品
