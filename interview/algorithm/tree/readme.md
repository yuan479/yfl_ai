# 二叉平衡树的数据结构特点

*树*
相对于链表和栈的线性结构，它是一种非线性的层次化数据结构

*二叉树*
递归定义：一个根节点由左子树和右子树组成，每一个子树也是一棵二叉树

*满二叉树*
除了叶子节点之外，每个节点都有两个子节点，且所有的叶子节点都在同一层

*完全二叉树*
除最后一层外，其他层完全填满，并且最后一层的节点也全都靠左排列。

*二叉搜索树（BST）*
对于任意节点，其左子树所有节点值小于该节点值，右子树所有节点值大于当前节点值，非常适合高效查找，插入，删除有序数据

*二叉平衡树*
基于二叉搜索树，约束左右子树的高度差，使其左右字数的高度差不超过 1 
二叉搜索树，为了查询的性能提升，O(n) -> O(log2 n)
O(n)：线性的时候，此时也没必要用树了，为了平衡一下，有了二叉平衡树
右旋或左旋达到平衡

如：   1                              3
         2                          /   \
           3         -->           2     4
             4                    /       \
               5                 1         5
AVL 树：叶子节点的层级相差不超过 1，为了查询效率而来，基于BST的优化。
