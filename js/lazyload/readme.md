# 图片懒加载

- 图片一次性全部显示出来？
  但是有时候不需要一次性全部显示出来，而且一次加载大量图片，对后端不太友好

- <img src=""/>
  - 浏览器的下载线程
  - src http 应用层协议
  - ip 地址
  - 在网络路径之中传输，但是网络带宽是有限的
    高并发 同时下载多个css，img 支持比较好的
    tcp/ip
  - 网页（电商）太多了
  - 滚动页面加载
  - 一开始如果src全部加载，页面会打不开

- 懒加载
  - 只加载需要加载的
    - 加载什么东西？ 可视区的内容
    - 滚动区域 scroll事件
  - 不加载
    - src 不能直接给， data-original 自定义属性
